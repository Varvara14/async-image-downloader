# Async Image Downloader

Асинхронное CLI-приложение для загрузки изображений с использованием `asyncio`, `aiohttp` и асинхронного ввода/вывода.

Проект демонстрирует:
- asynchronous I/O;
- конкурентную загрузку файлов;
- работу с asyncio queues;
- async workers;
- обработку HTTP-запросов;
- асинхронную запись файлов;
- обработку ошибок сети и ввода.

---

# Возможности

- одновременная загрузка нескольких изображений;
- асинхронная обработка URL;
- проверка корректности ссылок;
- автоматическое определение имени файла;
- обработка HTTP-ошибок и таймаутов;
- сохранение изображений в выбранную директорию;
- итоговая таблица успешных и неуспешных загрузок.
